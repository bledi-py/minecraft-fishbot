# Minecraft Automation (Python)

Small Python project to experiment with real-time screen capture
and simple UI automation.

Minecraft is used as a visual test environment.

## What it does
- Captures a small area of the screen
- Detects movement
- Triggers a mouse click when movement is detected

## Tech
- Python
- OpenCV
- MSS
- PyAutoGUI

## Notes
This is a hobby project and not meant to be perfect or production ready.
